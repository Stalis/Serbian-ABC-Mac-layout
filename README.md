# Serbian ABC keyboard layout for Mac OS

This is a keyboard layout for Serbian language, but based on English layout.
For people, that learning Serbian and used to standart ABC English keyboard.

# Layout

| Opt + | plain | with Shift |
|-------|-------|------------|
| s     | š     | Š          |
| d     | đ     | Đ          |
| z     | ž     | Ž          |
| c     | ć     | Ć          |
| x     | Č     | Č          


# Installation

1. Download latest dmg from Releases and open it
1. Open Keyboard Installer app
1. Drag ABC Serbian.bundle to the bottom of the installer window
1. Click on the method you want to install
1. Reboot
1. Select the layout in the Serbian (Latin) -> ABC Serbian section
